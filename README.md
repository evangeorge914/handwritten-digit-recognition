# Handwritten Digit Recognition using CNN

## Objective
To build a Convolutional Neural Network (CNN) model to recognize handwritten digits using the MNIST dataset.

## Dataset
- MNIST dataset (70,000 grayscale images of digits 0–9)
- Image size: 28 × 28 pixels

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

## Approach
1. Load and preprocess MNIST dataset
2. Normalize image data
3. Build CNN model with convolution and pooling layers
4. Train and evaluate the model
5. Predict handwritten digits

## Results
- Achieved ~98–99% accuracy on test data

pip install -r requirement.txt
